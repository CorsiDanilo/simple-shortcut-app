# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-28

### Added
- Create home screen shortcuts with a custom name and Material icon.
- Select multiple installed apps for each shortcut.
- Tap a shortcut on the home screen to open a fast, transparent picker dialog with your selected apps.
- Edit existing shortcuts (name, icon, app list) from the main app interface.
- Re-pin shortcuts to the home screen from the long-press menu.
- Delete shortcuts.
- Settings screen with language selector (Italian / English / System default).
- In-app update checker via GitHub Releases.
- Seamless OTA APK installation for updates.
- Changelog viewer inside the Settings screen.
