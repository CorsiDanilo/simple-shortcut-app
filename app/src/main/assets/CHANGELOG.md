# Changelog

## [1.0.0] - 2026-06-28

### Added
- Create home screen shortcuts with a custom name and Material icon
- Select multiple apps for each shortcut
- Tap a shortcut on the home screen to open a picker dialog with your selected apps
- Edit existing shortcuts (name, icon, app list)
- Re-pin shortcuts to the home screen from the long-press menu
- Delete shortcuts
- Settings screen with language selector (Italian / English / System default)
- In-app update checker via GitHub Releases
- Changelog viewer in Settings
